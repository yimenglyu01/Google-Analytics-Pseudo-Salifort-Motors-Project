# Salifort Motors Employee Attrition Prediction

This project, part of the *Google Advanced Data Analytics Capstone*, predicts employee attrition at **Salifort Motors** using machine learning techniques.

## Overview
The goal is to identify employees most at risk of leaving and provide actionable insights to improve retention and engagement.

The final model — **Random Forest v2** — achieved the best performance among all tested algorithms.

## Key Results
| Metric | Score |
|---------|-------|
| Accuracy | 0.955 |
| Precision | 0.850 |
| Recall | 0.883 |
| F1 Score | 0.866 |

**Top Predictors:**  
Number of Projects, last evaluation score, years worked in the company, and whether work longer than average standard.

## Business Insights
- Overworked or under-recognized employees show higher churn risk.  
- Improving workload balance and promotion transparency can reduce attrition.  
- The model can serve as an HR alert tool to identify at-risk employees early.

## Tools and Methods
Python · scikit-learn · pandas · seaborn · GridSearchCV · Random Forest
